# ML Projects

This folder contains applied machine learning and data science projects focused on solving real-world problems using statistical analysis, machine learning models, and data-driven decision making.

Each project includes exploratory analysis, model development, and documented findings.

## Projects

| Project                            | Description                                                                                        |
| ---------------------------------- | -------------------------------------------------------------------------------------------------- |
| **Yulu Demand Hypothesis Testing** | Statistical analysis of bike rental demand using hypothesis testing and exploratory data analysis. |
| **Jamboree Admission Prediction**  | Predictive modeling to estimate graduate admission chances using academic and profile features.    |

## Tools Used

Python • Pandas • NumPy • SciPy • Scikit-learn • Matplotlib • Seaborn
