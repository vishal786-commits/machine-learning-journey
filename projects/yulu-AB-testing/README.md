# Yulu Bike Rental Demand — Hypothesis Testing

Statistical analysis of bike rental demand using hypothesis testing and exploratory data analysis.

The objective of this project is to understand which factors significantly influence bike rental demand using statistical testing.


---

## Problem Statement

Yulu operates a micro-mobility service offering shared electric bikes.  
Understanding the factors that drive demand is critical for operational planning.

This project investigates whether different conditions significantly affect rental demand.

Key questions:

- Do working days have higher rentals than non-working days?
- Does weather significantly affect demand?
- Do seasons impact rental patterns?

---

## Methodology

The analysis follows a structured statistical workflow:

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Hypothesis Formulation
4. Statistical Testing

Statistical tests used:

- Two-Sample T-Test
- ANOVA
- Chi-Square Test

---

## Key Findings

- Working days have significantly higher rentals.
- Weather conditions strongly influence demand.
- Seasonal variation affects usage patterns.

These findings can help optimize:

- Fleet allocation
- Pricing strategies
- Operational planning
